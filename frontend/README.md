# Frontend Directory

This directory contains all client-side code and components.

## Suggested Structure

```
frontend/
├── components/     # Reusable React components
├── hooks/          # Custom React hooks
├── contexts/       # React context providers
├── types/          # TypeScript types/interfaces (client-side)
├── utils/          # Client-side utility functions
├── styles/         # CSS modules or styled components
└── constants/      # Client-side constants
```

## What Goes Here

- React components (buttons, forms, cards, etc.)
- Custom hooks (useAuth, useFetch, etc.)
- Context providers for state management
- Client-side utilities and helpers
- Type definitions for props and state
- Styling files specific to components
